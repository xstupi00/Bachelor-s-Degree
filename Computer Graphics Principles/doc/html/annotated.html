Data Structures
Here are the data structures with brief descriptions:
BunnyVertexThis structure represents vertex that contains only position and normal
GPUFragmentAttributesThis struct represents fragment that is produced by rasterization. Each fragment contains fragment attributes. Each attribute is represented as array of bytes that has to be cast to requested type
GPUFragmentShaderInputThis struct represents input data to fragment shader
GPUFragmentShaderOutputThis struct represents output data of fragment shader
GPUPrimitiveThis struct represents primitive - output of primitive assembly unit. A primitive is composed of N vertices, interpolation type for each vertex attribute and type of each vertex attribute
GPUTriangleThis structure represents triangle in clip-space. This structure is used in clipping. Positions of vertices are in clip-space. Barycentric coords are in respect to original triangle (output of primitive assembly unit)
GPUTriangleListThis structure represents triangle list - input and output of clipping
GPUVertexPullerConfigurationConfiguration for vertex puller
GPUVertexPullerHeadThis struct represents configuration for one reading head of vertex puller
GPUVertexPullerOutputThis struct represents output of vertex puller - vertex
GPUVertexShaderInputThis struct represents input interface of vertex shader
GPUVertexShaderOutputThis struct represents vertex that is output of vertex shader
Mat4This struct represents 4x4 matrix - it is represented by 4 column vectors
PhongVariablesThis structure contains all global variables for this method
TriangleExampleVariables[EXTERNS]
Vec2This struct represents column 2D vector of 32-bit floats
Vec3This struct represents column 3D vector of 32-bt floats
Vec4This struct represents column 4D vector of 32-bit floats
