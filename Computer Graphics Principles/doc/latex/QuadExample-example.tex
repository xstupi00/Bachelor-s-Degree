Proměnné \char`\"{}uniform\char`\"{} slouží pro uložení konstant, které zůstávají stejné po dobu jednoho vykreslujícího příkazu. \char`\"{}layout(location=n)\char`\"{} explicitně specifikuje lokaci dané poměnné."\par
