\subsection{Detailed Description}
This file contains structures and function declaration that are needed for manipulation with uniforms. \begin{DoxyAuthor}{Author}
Tomáš Milet, \href{mailto:imilet@fit.vutbr.cz}{\tt imilet@fit.\-vutbr.\-cz}
\end{DoxyAuthor}
All uniform values are constants that can only be read on G\-P\-U (in shaders). A uniform value does not change during drawing. A uniform value can be changed from C\-P\-U side between draw calls. They are useful for storing matrices, samplers and other global constants.
